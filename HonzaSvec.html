Zde by časem mohl vyrůst archiv mé dosavadní tvorby pro PyCZ. Pro začátek určitě vezmete za vděk alespoň následujícími odkazy:

  • Učebnice jazyka Python je překlad originálního tutorialu jazyka Python ve verzi 2.2. Originální tutorial je napsán Guidem van Rossumem a Fredem L. Drakem. V roce 2002 jsem se rozhodl přeložit tento tutorial jako pomoc všem dalším začínajícím programátorům v jazyce Python.
  • http://www.pythondocs.ic.cz/tut/index.html
  • http://geon.wz.cz/docs/
  • http://macek.sandbox.cz/texty/python-tutorial-cz/tut

  • DoporuceniProPreklad odborných textů v jazyce Python - materiál, který jsem sestavil při překládání výše uvedené učebnice jazyka Python.

  • Překlad rozhovoru s autorem Pythonu Guidem van Rossumem, rozmlouvá s ním Bill Venners - originál najdete na http://www.artima.com/intv/guido.html .

    • RozhovorCast1 - Historie Pythonu
    • RozhovorCast2 - Cíle a návrh Pythonu
    • RozhovorCast3 - Rapidní vývoj v Pythonu
    • RozhovorCast4 - Závazky v programování
    • RozhovorCast5 - Statické vs. dynamické typování
    • RozhovorCast6 - Veřejné API jazyka Python

Moje dlouhodobá práce pro server ROOT.cz:

  • Seriál Létající cirkus (díky Lukášovi Tinklovi i PDF verze) - pojednává o úvodu do jazyka Python i do jeho modulů jako jsou os, pickle, urlib, rexec, pil a dalších.
  • Seriál Make-up pro vaše aplikace úvod do programování grafického uživatelského rozhraní pomocí PyGtk, PyQt a Tkinter.
  • Dvojdílný článek Nový chřestýš novinky, jež byly uvedeny ve verzi 2.4 jazyka Python, tj. generátorové výrazy, dekorátory, datový typ množina, moduly subprocess a doctest. (Pozn. v názvu článku vystopujete moje zmatení smyslů: python != chřestýš, nýbrž krajta, pokud to dobře chápu).

Další články pro ROOT.cz, které se již přímo netýkají jazyka Python, ale mají mnoho společného buď s operačním systémem Linux nebo dalším svobodným software:

  • Dvojdílný článek Upravte si klávesnici - jak si pod XFree86 (a dalšími X servery s XKB extenzí) upravit definici klávesnice.
  • Článek Hlídejte si instalaci o vytváření binárních balíčků na linxových distribucích pomocí skriptu checkinstall.
  • Několik rozhovorů pod společným názvem Zpovědnice - Pavel Satrapa, Lukáš Tinkl, Tým CZilla, Josef Perzyna.
