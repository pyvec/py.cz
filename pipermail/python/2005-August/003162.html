Staci toto:
apt-get install mod_python (alebo podla distra)

v apache2 conf adresare: 
cat mod_psp.cfg
 AddHandler mod_python .psp
 PythonHandler mod_python.psp
 PythonDebug Off

linku z mods_available do mods_enabled (mod_python.load)
a restarnut apacha

/etc/init.d/apache2 restart

Skusis napr.:
cat skuska.psp
def pr(tlac="<h2>Ziadny parameter neprisiel do funkcie \'pr\'.</h2>"):
 req.write(str(tlac))
pr ("skuska tlace z pythona")
