Dobry den,

snazim se cist potrebne informace z registru windows za pomoci Python 2.
Delam to nasledujicim zpusobem:
import re
from _winreg import *
Patch = OpenKey(HKEY_LOCAL_MACHINE, r'Software\Rockwell Software\RSLinx\Last Version', 0, KEY_ALL_ACCESS)
RSLinx = (QueryValueEx(Patch, "Base Version"))

Chtel bych ale vedet jak cist, ulozit nazvy slozek v registrech.
Napriklad cesta Software\Rockwell Software\RSLinx\Last Version a v ni je jeste nekolik podslozek, ktere nelze predem definovat, jejich nazev nemusim znat. A ja chci precist hodnotu "Base Version" z kazde teto slozky.
