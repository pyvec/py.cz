livewires po instalaci zrejme nebude na PYTHONPATH

Overeni, zda tam je, nebo neni:

 - pustit interaktivni shell Pythonu (nebo jeste lepe IPythonu)
 - import sys
 - sys.path

To vypise veskere adresare, kde Python hleda pro moduly pri importu
(modul = podslozka s init.py a nebo samostatny python soubor).

Pokud livewires ve vypsanych slozkach neni, pak to Python nemuze importovat.

Potom je potreba opakovat instalaci, nebo nakopirovat rucne cely modul
livewires nekam do Python cesty, nebo pouzit ve slozce Pythonu
nejakysoubor.pth pro nastaveni cesty k modulu, nebo nasetovat promenou
postredi PYTHON_PATH na cestu k modulu.
