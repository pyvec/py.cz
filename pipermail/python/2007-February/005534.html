Zdravim,

promenou arr tridy cSet jsi definoval jako tridni (statickou) promennou. 
Tridu cSet staci definovat takto:
class cSet:
    def __init__(self):
       self.arr = []

Jeste si neodpustim poznamku, ze list je build-in funkce, takze 
nedoporucuji pouzivat jako promennou.
