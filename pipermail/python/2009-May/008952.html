> .pyw je zdrojak v Pythonu pro Windows GUI (sam jsem to nikdy nevidel)
pyw je ve Windows volan pomoci pythonw.exe, pouziva se to kdyz mas GUI 
aplikaci a nechces aby se uzivateli pri spusteni zobrazilo spolecne s 
programem i konsolove okno.

Takze program se nejdrive ladi jako normalni .py a jakmile je odladen, 
prejmenuje se na .pyw.

Zkusit si to muzes treba na tomhle prikladu:

from sys import exit
from Tkinter import *
root = Tk()
print 'Tohle se vytiskne jen kdyz to spustis jako .py'
Button(root, text='Klikni sem pro ukonceni!', command=exit).pack()
root.mainloop()

Jednou to spust jako .py, pak to prejmenuj na .pyw a uvidis rozdil.

Funguje to jenom na Windows.
