    Specifying additional files

Some applications need additional files at runtime, like configuration 
files, fonts, or bitmaps.

*py2exe* can copy these files into subdirectories of dist if they are 
specified in the setup script with the data_files option. data_files 
should contain a sequence of (target-dir, files) tuples, where files is 
a sequence of files to be copied.

Here's an example:

# setup.py
from distutils.core import setup
import glob
import py2exe

setup(console=["myscript.py"],
      data_files=[("bitmaps",
                   ["bm/large.gif", "bm/small.gif"]),
                  ("fonts",
                   glob.glob("fonts\\*.fnt"))],
)
