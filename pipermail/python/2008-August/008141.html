Musite volat aj konstruktor predka. Robi sa to bud:

class Child(Base):
     def __init__(self):
         Base.__init__(self)
         self.x = 1

alebo:

class Child(Base):
     def __init__(self):
         super(Child, self).__init__()
         self.x = 1
