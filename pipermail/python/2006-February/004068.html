In [1]: import re

In [2]: re.match("^(.)(.)$", "aa") != None

In [3]: re.match("^(.)(.)$", "aa").groups()

Out[4]: True
In [5]: re.match("^(?:.)(?:.)$", "aa").groups()
Out[5]: ()

expression is inside the parentheses, but the substring matched by the
