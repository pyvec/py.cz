Dobry den,

RedHat 7.3 v sobe sice dokumentaci Pythonu 2 ma, ale v nejakem LaTexu,
je potreba si HTML dodelat. Mne se to povedlo bez problemu, tipnul bych
si, ze se spousti Python 1.5x, zatimco skripty na sestaveni HTML verze jsou
delany pro 2.x. Cesta nejmensiho odporu podle me spociva ve vytvoreni
symlinku /usr/bin/python -> /usr/bin/python2. Po uspesnem sestaveni je mozne
to dat zase zpatky, aby prikaz python spoustel verzi 1.5x. Dalsi moznost je
ve skriptech predelat vsechny "hashbangy", aby napr. (nevim, jak to je
presne)
misto #!/usr/bin env python bylo #!/usr/bin/env python2.
