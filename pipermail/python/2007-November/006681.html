Ahoj,
Nevim uplne co myslis tim prohledavanim, ale jestli myslis jen jak si
vyhodit text v zavorkach pro pozdejsi pouziti, tak pak treba tenhle
cyklus:

s = 'abc{tohle prohledej}fgth12'
for i in range(len(s)):
    if s[i] == '{': zacatek = i+1
    if s[i] == '}':
        konec = i
        break


print s[zacatek:konec]
