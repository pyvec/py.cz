Tak z vyberu Python nebo VB je Python asi ta lepsi volba. Ale IMHO na
vyuku programovani (jen zduraznuju, ze vidim rozdil mezi programovanim
a algoritmizaci), neni Python nejlepsi jazyk. Mozna jsem prilis
starosvetsky, ale je potreba vedet, jak veci funguji uvnitr, aspon
trochu, a na to ma python moc vysokou uroven abstrakce. Navic kdyz
vezmu jake zmateni dokazou udelat mutable/immutable typy predavane
jako parametr funkce... No zlaty Pascal :)

Ale samozrejme zalezi na tom, co ma byt cil toho predmetu - jestli
naucit decka aspon trochu "programatorsky myslet", pak je to Python
all the way up. Ale bal bych se nekoho ucit na Pythonu "programovani".
