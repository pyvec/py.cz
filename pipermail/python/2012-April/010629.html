Ahoj,

me to teda funguje i se zvyraznovanim syntaxe, ale nemam to ve vimrc,
ale v ~/.vim/ftplugin/python.vim
v .vimrc mam

set fileencodings=utf-8,cp1251,default,latin1

filetype indent plugin on

set list listchars=tab:>_,trail:.

au filetype python set omnifunc=pythoncomplete#Complete

set wildmenu

colo elflord

syntax on

setlocal expandtab
setlocal tabstop=4
setlocal softtabstop=4
setlocal shiftwidth=4


IMHO by ti melo stacit do .vimrc pridat "syntax on" a bude to v cajku.
Pripadne to nastaveni pro Python presunout do .vim/ftplugin/ a pridat
do .vimrc "filetype indent plugin on"
