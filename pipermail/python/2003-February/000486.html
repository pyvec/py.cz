Ahoj,
jak jsem tu jednou psal, tak jsem vytvoril gui framework pro
Python. Jestli mate naladu, stahujte z
http://dev.py.cz/user/honza/igui2.tar.gz a zkompilujte
(potrebujete GTK+ 1.2). Pak muze vase aplikace v Pythonu mit
treba takovej zdrojak:


#!/usr/bin/env python2.2

from igui.classes import *

app = Application(name='hello_world')
win = Window(title='Hello world')
win.packing_manager.border = 15

win.append(Button(name='button', title='Pozdrav\nsvet!'))
def print_hello(e):
	print 'Hello world'
win.button.OnActivate.append(print_hello)

app.Run(win)


Pozor, je silne devel, mnoho veci neni dodelanych kazdopadne
prvni jakz-takz funkci verze behem pristich tri tydnu. Zbyva
dodelat generovani nekterych udalosti atd.

Pokud mate naladu, otestujte a sdelte mi sve pocity.
