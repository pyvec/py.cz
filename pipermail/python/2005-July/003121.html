Ahoj,
  od verze 2.3 podporuje Python knihovnu sets, coz
jsou  netridene seznamy unikatnich polozek. Takze
podle mne nejjednodussi je toto:

import sets

set =sets.Set(seznam)  // seznam obsahuje AAA, BBB,
atd

//vypis polozek
for polozka in set:
  print polozka

A to je vse. Navic sety podporuju matematicke operace
jako sjendoceni, prunik atd.
