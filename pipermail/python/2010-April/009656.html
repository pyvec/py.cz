Este ma napadlo jedno riesenie. Ale rovnako blbe ako vsetky ostatne :) 
Len je ciste, bez metaprogramovania.

class tridaA():
     @staticmethod
     def _b(self, x):
         self.x=x

     def b(self, x):
         return self._b(self, x)


a potom:

a = tridaA()
a._b = c

a bude fungovat.
