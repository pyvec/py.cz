 Je trochu skoda ze vetsina uvedenych prikladu (nebo dokonce vsechny)
pouzivaji knihovnu ktera se musi stahnout z jejich webu a neni free :-/

  Ciste v Pythonu jsou knihovny md5 a sha, v pythonu 2.5 a novejsim jsou obe
hashovaci funkce spolecne v modulu hashlib.

Priklad:
D:\>python
Python 2.4.4 (#71, Oct 18 2006, 08:34:43) [MSC v.1310 32 bit (Intel)] on
win32
>>> import md5
>>> test = 'nejaky string'
>>> hash = md5.new(test)
>>> hash.hexdigest()
'0120e0ca1d64cf32c27dcf6b65bf8922'
>>> import sha
>>> sha_hash = sha.new(test)
>>> sha_hash.hexdigest()
'7d14dc4849b201b7c63a0db3263be50d0481737e'
>>>

Na slozitejsi veci se hodi PyCrypto (musi se stahnout) a jeho "high-level
interface" EzPyCrypto (taktez)
