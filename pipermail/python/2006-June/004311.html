Zkusil bych nastavit option databazi:

V pripade, ze program pouziva MenuBar z MegaWidgets treba takto
root = Tk()
root.option_add('*MenuBar*Font', 'Arial 12')


Nebo Tk vyuzivajici klasicke menu:

root = Tk()
root.option_add('*Menu*Font', 'Arial 12')
root.option_add('*Menubutton*Font', 'Arial 12')

ten arial si uprav dle OS ;-)
