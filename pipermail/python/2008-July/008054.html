>> > binarnich souboru s vasim kodem (typicky vami vytvorene balicky,

>> > zdrojovy tar.gz balik, rpm balik, Windows self-extract balik, a dokonce
>> > existuje i Nullsoft NSIS plugin, takze muzete vytvaret i Windows
