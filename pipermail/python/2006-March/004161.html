Dobry den,
pokusim se neco psat za pouziti pyserial. V pythonwin nebo v linuxu 
interaktivnim pythonu mi to funguje. Pokousimli se ale spustit script 
dostanu hlaseni :

c:\data\serial>python gregor.py
Traceback (most recent call last):
   File "gregor.py", line 2, in ?
     import serial
   File "c:\data\serial\serial.py", line 2, in ?
     ser = serial.Serial(0)
AttributeError: 'module' object has no attribute 'Serial'

Podobne v linuxu. Pritom podle stranek projektu by takhle melo fungovat. 
Netusite nekdo v cem by mohl byt problem ? Dekuji za nakopnuti :-)

import serial
ser = serial.Serial(0)
ser.setDTR(1)
print ser.getDSR()
ser.close()
