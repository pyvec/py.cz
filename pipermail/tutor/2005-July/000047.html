Potrebuji nahradit vybrane znaky jinymi.
Hledam funkci, ktera je ekvivalent funkce strtr() z PHP. Na nic
podobneho jsem v helpu nenarazil. Poradte, kde ji mam hledat?
Zatim jsem si napsal funci vlastni, ale radsi bych pouzil nejakou z
modulu.

def strtr(text, cfrom, cto):
        'Replace chars strtr(text, characters-from, characters-to) -> string'
        t=''
        for c in text:
                if c in cfrom:
                        t+=cto[cfrom.index(c)]
                else:
                        t+=c
        return t

>>> s="text <text> text"
>>> strtr(s,'<>','[]')
'text [text] text'
