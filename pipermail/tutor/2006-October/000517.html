Postup na kompilaciu do exe:

1. Stiahni py2exe z http://www.py2exe.org/

2. Nainstaluj py2exe

3. Napis si takyto podobny skript s nazvom setup.py
------------------------------------------------------------------
# setup.py
from distutils.core import setup
import py2exe

setup(windows=["wsrbistat.py"])
------------------------------------------------------------------
wsrbistat.py tam nahrad menom svojho programu.


4. umiestni svoj program a setup.py do toho isteho adresara a zadaj na 
commandline:

python setup.py py2exe

ak chces help zadaj: 

python setup.py py2exe --help

5. Po skompilovani ti v tvojom adresari vzniknu podadresare
build
dist
V adresari dist mas exe svojho programu a ostatne komponenty ktore musis 
distribuovat.
