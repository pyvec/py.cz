StringIO je takovy "soubor v pameti" - neni treba zakladat soubor nekde v
tempu

externimu programu predhodit tento objekt, aby jej pouzil jako vystup..


tak, ze vyuzijes parametry funkce execfile(), ktere jsou locals a
glovbals,globals napr. takto:

from StringIO import StringIO
import sys
oldout = sys.stdout
myout = StringIO()
sys.stdout = myout
d = {'__name__':'__main__', 'sys':sys}
execfile('ex.py', d, d)
sys.stdout = oldout
myout.seek(0)
print myout.buf


no a myout.buf nyni obsahuje vse, co program ext_program vyslal na
standartni vystup
