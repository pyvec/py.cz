Python je skvělý programovací jazyk pro začátečníky i pro pokročilé, pro studenty i pro vydělávající programátory. Programy se v něm tvoří rychleji než v mnoha jiných jazycích i proto, že se dodává s rozsáhlou knihovnou modulů. Proto si Python oblíbili programátoři klasických i webových aplikací. Správcům systémů nabízí Python lepší možnosti než klasické interprety skriptů.

Download a Instalace

Na webu http://www.python.org/ najdete verze pro operační systémy Windows, Linux, Mac OS X a další.

  • 2.x i 3.x stahujte na http://www.python.org/download/
  • Jak instalovat: InstalaceWindows, InstalaceLinux

Jaký používat textový editor

Pro absolutní začátečníky snad nejvhodnější IDLE - editor dodávaný s instalací nebo nějaký lehkotonážní editor, podporující barevnou syntaxi Pythonu. O mnoho více najdete v Editory-IDE.

Volba grafického prostředí

Umíte již základy programovacího jazyka Python a taky trochu anglicky? A chtěli byste začít tvořit okenní aplikace? Potřebujete tedy nějaké GUI. Python jich vám nabízí dost: GrafickaProstrediGUI anebo ještě více: http://wiki.python.org/moin/GuiProgramming. Abychom vám usnadnili volbu, připravili jsme pro vás list JakeZvolitGUI.

Máte nějaké otázky?

Něco nefunguje jak má, něco není z tutoriálů a příruček jasné, něco jste tady na webu prostě nenašli? Přihlaste se do KonferenceDiskuze, pošlete dotaz a někdo vám jistě poradí.

Oblíbili jste si Python?

Podělte se o své zkušenosti. Je velmi pravděpodobné, že podobný problém bude mít i někdo jiný. Přispějte svou znalostí do databáze této wiki. Přispěvatelem může být doslova každý. Děkujeme za váš čas i pečlivost při zápisu. HelpPage
